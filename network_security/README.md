# Network Security Lab

This folder contains basic implementations of cryptographic algorithms and security concepts using Java.

## Programs

* Caesar Cipher – Basic substitution cipher
* AES Encryption – Symmetric encryption using AES
* DES Encryption – Basic symmetric encryption
* XOR Encryption – Simple encryption for demonstration

## Technologies

* Java
* javax.crypto

## How to Run

1. Compile:
   javac FileName.java
2. Run:
   java FileName

## Note

These programs are implemented for learning purposes and demonstrate fundamental concepts of cryptography. They are not intended for real-world security use.
