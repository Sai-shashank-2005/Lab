# Web Technology Lab

This folder contains basic HTML programs demonstrating core web development concepts.

## Programs
- colspan.html – Table column span example
- rowspan.html – Table row span example
- newspaper.html – Layout design
- number_to_word.html – Number conversion logic
- payroll_system.html – Simple payroll calculation
- result_system.html – Student result processing

## How to Run
Open any `.html` file in a browser.
